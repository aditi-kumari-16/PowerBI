E-Commerce Sales Analysis | Amazon Sales Report

Project Overview
Analyzed Amazon e-commerce sales data to uncover key business insights and improve decision-making. The project focuses on revenue trends, category performance and customer behavior through interactive dashboards.

Tools & Techniques
- Tools: Excel, Power BI  
- Skills: Data Cleaning, Visualization, KPI Analysis, Business Insights  

Key Insights
- Processed 1,20,000+ sales records, removed duplicates and missing data for accuracy.  
- Generated total revenue of ₹73.41M with the following KPIs:
  - Success Rate: 83.37%
  - Return Rate: 1.65%
  - Cancellation Rate: 14.28%
- Top State: Maharashtra (₹12M)  
- Top Category: Sarees (₹37M, ~50% of total revenue)  
- Peak Month: April (~₹20M revenue)  
- 69% of orders were expedited and 99.9% of sales occurred via Amazon.
